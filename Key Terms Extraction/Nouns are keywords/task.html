Theory

The output from the last stage looks so much better! We can still improve the results, though: adverbs and adjectives don't always reflect the essence of the text. Nouns are very useful with this. In this stage, we will improve the result by using part-of-speech tagging (a POS-tag) to mark nouns as keywords.

Description

The task is similar to the previous ones: tokenize, eliminate meaningless words, and lemmatize. In addition to removing punctuation and stop words, remove all the non-noun words. You can use averaged_perceptron_tagger for this. You need to load it first:

nltk.download('averaged_perceptron_tagger')

Then, let's POS-annotate the pre-lemmatized text with the function below. Be sure that you tag the whole news article, not just one word or sentence.

pos_tag(lemmatized_text)

And, voil√†, we get POS tags. Now, sort your text so that there are only tokens with NN in the result file.

If the frequencies for some tokens are the same, then sort them in descending order. For example, for the following dictionary, where the keys are the names of fruits, and the values of the frequency of their occurrence:

{"apple": 5, "lemon": 6, "banana": 5, "grapefruit": 1, "pineapple": 5, "watermelon": 3, "melon": 4, "dragonfruit": 5}

After sorting, the dictionary should be

{"lemon": 6, "pineapple": 5, "dragonfruit": 5, "banana": 5, "apple": 5, "melon": 4, "watermelon": 3, "grapefruit": 1}

Take a look at the attached news file and use it at this stage.

Objectives

In this stage, your program should:

  1. Read an XML-file containing news stories and headlines.
  2. Extract the headers and the text.
  3. Tokenize each text.
  4. Exclude the tokens that start with ', stopwords, and punctuation marks.
  5. Lemmatize the remaining tokens in the story. Take into consideration only those lemmas, the length of which is bigger than 1.
  6. For each news story, find the most frequent nouns.
  7. Print each story's headline and the five most frequent nouns in descending order. Take a look at the sample below. Display the titles and keywords in the same order they are presented in the file.
