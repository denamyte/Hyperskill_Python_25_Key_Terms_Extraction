Theory

We need to improve the results because they don't really look like keywords. In this stage, get rid of function words, punctuation marks, and lemmatize the text.

Articles, conjunctions, and some other words belong to the stopwords group. These words do not have a particular meaning, but they are extremely common in text and can influence the result significantly. In the nltk library, you can get a list of these words by using:

stopwords.words('english')

You can import the stopwords in the following way:

from nltk.corpus import stopwords

How can we get rid of punctuation? There are plenty of methods: from iterating through every character to using regular expressions. In this project, we suggest using the string library.

Finally, lemmatize the words using WordNetLemmatizer() from the nltk library. By default, the WordNetLemmatizer lemmatizes only nouns.

In this stage, don't lemmatize words besides nouns.

Once you have the summaries for each piece of news, you should print them as you did in the first stage.

Description

In this stage, you need to tokenize the text, get rid of punctuation marks and stopwords, and lemmatize the text. Then, calculate the frequency once again and output the five most frequent words. Don't forget to convert the text to lowercase before tokenization:

nltk.tokenize.word_tokenize(text.lower())

Mind that the word_tokenize() function might identify such contractions as 's as separate tokens. You need to exclude these tokens, too. To identify them, you can use the startswith() function.

If the frequencies for some tokens are the same, then sort them in descending order. For example, for the following dictionary, where the keys are the names of fruits, and the values of the frequency of their occurrence:

{"apple": 5, "lemon": 6, "banana": 5, "grapefruit": 1, "pineapple": 5, "watermelon": 3, "melon": 4, "dragonfruit": 5}

After sorting, the dictionary should be

{"lemon": 6, "pineapple": 5, "dragonfruit": 5, "banana": 5, "apple": 5, "melon": 4, "watermelon": 3, "grapefruit": 1}

Mind the output formatting! You need to follow it so that the system can check the solution. Display the titles and keywords in the same order as the news items in the file.

Take a look at the attached news file and use it in your project.

Objectives

Your program should:

  1. Read an XML file containing news stories and headlines.
  2. Extract the headers and the text.
  3. Tokenize each text.
  4. Exclude the tokens that start with ', stopwords, and punctuation marks.
  5. Lemmatize the remaining tokens in the story. Take into consideration only those lemmas, the length of which is bigger than 1.
  6. For each news story, find the most frequent lemmas.
  7. Print each story's headline and five most frequent tokens in descending order. Take a look at the sample below. Display the titles and keywords in the same order they are presented in the file.
